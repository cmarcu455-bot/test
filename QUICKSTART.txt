🚀 FASTEST SETUP (10 minutes):

STEP 1: Fill Excel File
───────────────────────
1. Open: Apartment_Sales_Template.xlsx
2. Go to "Apartments" sheet
3. Replace 9 sample apartments with YOUR 249 apartments
4. Fill in all columns: apartment_number, bedroom_type, base_price, parking_spots
5. Add all your styling options to "Styling_Options" sheet (flooring, tiling, etc.)
6. Add all your buyers to "Buyers" sheet
7. Add current selections to "Buyer_Selections" sheet
8. Save as Excel file

STEP 2: Deploy App (Choose ONE):
────────────────────────────────

Option A - EASIEST (Online Hosting):
  1. Go to: vercel.com or netlify.com
  2. Upload: app.html, sw.js, manifest.json
  3. Get URL instantly
  4. Share with your team
  ✓ Best for multiple teams

Option B - LOCAL (Free, No Server):
  1. Download Python (if needed)
  2. Put app.html, sw.js, manifest.json in a folder
  3. Open terminal/cmd in that folder
  4. Run: python -m http.server 8000 (or python3 on Mac)
  5. Access from tablet: http://[YOUR-COMPUTER-IP]:8000/app.html
  ✓ Example: http://192.168.1.100:8000/app.html
  ✓ Works on same WiFi

Option C - OFFLINE ONLY:
  1. Download app.html to tablet
  2. Open in Chrome/Firefox browser
  3. Works completely offline
  ✓ Simplest but no sync between tablets

STEP 3: Use on Android Tablet
──────────────────────────────
1. Open app in browser (whichever option you chose)
2. Tap 3-dot menu → "Add to Home Screen"
3. Icon appears on home screen
4. Click "Data Management" tab
5. Click "Upload Excel File"
6. Select your filled Excel file
7. Click "Import Data"
8. Done! Now use the app

❓ COMMON QUESTIONS:

Q: Do I need internet?
A: First time: Yes (to cache app)
   After: No! Works completely offline

Q: Where does data go?
A: Stays on the tablet in browser storage. Never sent anywhere.

Q: How do I sync between 2 tablets?
A: 
   Tablet 1: Click "Export Data" → Save to OneDrive/Google Drive
   Tablet 2: Click "Import Data" → Select file from cloud
   Takes 2 minutes

Q: What if tablet breaks?
A: Export regularly to Excel. You have backups on cloud.

Q: Can I edit the app?
A: Yes! All code is in app.html. Easy to customize.

Q: What if Excel import fails?
A: Check Excel sheet names are exact (case-sensitive)
   Check column headers match
   Try saving Excel as .xlsx again
